1. This is a Matlab based interactive program to predict the steady-state positions of main drainage divide (MDD) within a mountain belt. 
2. This program is an upgraded version of the SDD-C program, with constraints of horizontal tectonic movement (advection) added on its basis.
3. The model accepts user-defined input parameters, including uplift rate, horizontal advection rate, erosion coefficient, mountain belt width, hillslope width, base-level elevation, and other relevant variables.
